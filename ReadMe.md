SnowDay
===

####Introduction
Is an Internet Enabled object that searches twitter for a hashtag, and affects a LED in response to the search.

####Components
* 1x Particle Photon
* 1x LED
* 1x Photoresistor

####Code
There are two parts to the project.

1. A Particle sketch
2. Some php scripts
